# **Bus Ticket Booking Portal**

The **Bus Ticket Booking Portal** is a user-friendly platform that simplifies bus ticket reservations for students and faculty. With real-time database updates, the system ensures accurate seat availability and smooth management of bookings.

## **Features**

- **Effortless Booking & Cancellation:** A streamlined process for booking and canceling tickets, with an intuitive interface for easy navigation.
- **Real-Time Database Updates:** Ensures the latest seat availability, with data synced in real-time.
- **Temporary Seat Lock:** To prevent double bookings, the system temporarily "locks" selected seats during the booking process.
- **Secure User Registration:** Uses **PHP Mailer** for email verification to ensure secure and authenticated user registration.

## **Technology Stack**

- **Backend:** PHP
- **Frontend:** HTML, CSS, JavaScript
- **Database:** MySQL
- **Email Verification:** PHP Mailer

## **How to Use**

1. **Sign Up:** Register using your email.
2. **Verify Email:** Complete the email verification process.
3. **Book a Ticket:** Choose your preferred bus and seat, then proceed to book.
4. **Manage Bookings:** View or cancel your bookings anytime through the portal.
